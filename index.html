The data analyst is responsible for taking actionable that affects the current scope of the company. The data engineer is responsible for developing a platform that data analysts and data scientists work on. And, the data scientist is responsible for unearthing future insights from existing data and helping companies to make data-driven decisions.

The data analyst does not directly participate in the decision-making process, rather, he helps indirectly through providing static insights about company performance. The data engineer is not responsible for decision making. And, The data scientist participates in the active decision-making process that affects the course of the company.

The data analyst uses static modeling techniques that summarize the data through descriptive analysis. On the other hand, The data engineer is responsible for the development and maintenance of data pipelines. The data scientist uses dynamic techniques like Machine Learning to gain insights about the future.

Knowledge of machine learning is not important for data analysts. However, this is mandatory for data scientists. The data engineer may not be required to have the knowledge of machine learning but he is required to have the knowledge of core computing concepts like programming and algorithms to build robust data systems.

The data analyst only has to deal with structured data. However, both data scientists and data engineers deal with unstructured data as well.

Data analysts and data scientists are both required to be proficient in data visualization. However, this is not required in the case of data engineering.

Both data scientists and analysts may not have knowledge of application development and working of the APIs. However, this is the most essential requirement for data engineering.
